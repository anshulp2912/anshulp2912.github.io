# Product Overview

This is a personal portfolio website with an integrated chatbot resume feature called "BOTAnshul". The project combines a traditional portfolio website showcasing Anshul Patel's professional experience, education, projects, and skills with an interactive Rasa-powered chatbot that serves as a conversational resume.

## Key Features

- **Portfolio Website**: Professional showcase with sections for about, experience, education, projects, skills, and contact information
- **Interactive Chatbot**: BOTAnshul - a Rasa-powered conversational AI that can answer questions about Anshul's background, projects, and experience
- **Responsive Design**: Mobile-friendly interface with modern web technologies
- **Heroku Deployment**: Containerized deployment using Docker and Heroku

## Target Audience

Potential employers, recruiters, and professional contacts who want to learn about Anshul Patel's background and qualifications through either traditional browsing or conversational interaction.
